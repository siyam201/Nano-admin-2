# Nano Admin

**Lightweight Admin Panel Generator**

Nano Admin হলো একটি শক্তিশালী, পুনঃব্যবহারযোগ্য Admin Panel যা আপনি যেকোনো ওয়েব অ্যাপ্লিকেশনের কন্ট্রোল সেন্টার হিসেবে ব্যবহার করতে পারেন।

## Core Features

### Authentication System
- Email + Password Login/Register
- JWT Access Token (15 min expiry) + Refresh Token (7 days)
- bcrypt Password Hashing
- Session Management

### Role & Permission
- **Admin**: Full access to all features
- **User**: Limited access based on role

### Dashboard
- Total Users count
- Active Users count
- New Users Today
- Recent Activity Feed
- System Status

### CRUD Engine
- Generic Table View with sorting
- Add / Edit / Delete operations
- Search functionality
- Pagination support

### API Keys
- Create API keys for external apps
- Enable/Disable keys
- Track last used time
- Secure key generation

## How to Connect External Apps

### Method 1: API-Based Connection

Use the generated API keys to connect your other applications:

```bash
# Example: Get all users from external app
curl -X GET "https://your-nano-admin.replit.app/api/users" \
  -H "Authorization: Bearer nano_your-api-key-here"
```

### API Endpoints

#### Authentication
```
POST /api/auth/register    - Register new user
POST /api/auth/login       - Login user
POST /api/auth/refresh     - Refresh access token
POST /api/auth/logout      - Logout user
PATCH /api/auth/profile    - Update profile
POST /api/auth/change-password - Change password
```

#### Dashboard
```
GET /api/dashboard/stats   - Get dashboard statistics
```

#### Users (Admin only for create/delete)
```
GET /api/users             - List all users (paginated)
POST /api/users            - Create new user
PATCH /api/users/:id       - Update user
DELETE /api/users/:id      - Delete user
```

#### Activities
```
GET /api/activities        - List all activities (paginated)
GET /api/activities/recent - Get recent activities
```

#### API Keys (Admin only)
```
GET /api/api-keys          - List all API keys
POST /api/api-keys         - Create new API key
PATCH /api/api-keys/:id    - Update API key
DELETE /api/api-keys/:id   - Delete API key
```

### Method 2: Same Database

If your main application uses the same PostgreSQL database, Nano Admin can directly manage the data. Just connect both apps to the same `DATABASE_URL`.

## Tech Stack

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **ORM**: Drizzle
- **Auth**: JWT (jsonwebtoken) + bcrypt
- **UI**: Shadcn/ui components

## Environment Variables

```
DATABASE_URL=your-postgres-connection-string
SESSION_SECRET=your-secret-key-for-jwt
```
